service_routes module¶

service_routes.handle_create_service(handler)[sorgente]¶

POST /services - Crea un nuovo servizio nel DB. Body JSON: {«name»:»…», «description»:»…», «capacity»:…, «price»:…, «active»:1/0}

service_routes.handle_delete_service(handler, service_id)[sorgente]¶

DELETE /services/<id> - Elimina il servizio dal DB (soft delete or real delete).

service_routes.handle_get_all_services(handler)[sorgente]¶

GET /services - Ritorna tutti i servizi dal DB.

service_routes.handle_get_service_by_id(handler, service_id)[sorgente]¶

GET /services/<id> - Ritorna il singolo servizio, se esiste.

service_routes.handle_update_service(handler, service_id)[sorgente]¶

PUT /services/<id> - Aggiorna i campi di un servizio.
